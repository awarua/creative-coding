          This lecture introduces you to DXB303

          W2 Mark Making, Drawing and Sketching

          The focus of this week's lecture is on mark marking, or drawing with code. We will look at how to use mouse input to create simple drawing sketches, how to vary the visual properties of the drawing, and how to save images from Processing.


          		 Active Drawing

          This is the first week of tutorials.  Please make sure you are signed up to a tutorial class AND that you attend the tutorial you are signed up for.

          From visual art to computer games:generative and procedural approaches. This week's lecture introduces covers iteration and pattern, and then there is external video on Generative Design.


          		 Making Patterns

          In this weeks tutorial we will look at using Processing to create Patterns exploring foundational programming concept of iteration (looping and repeating). 


          		    
          W4 Images, Arrays, and Colour

          This week's lecture covers working with Bitmaps in Processing, and introduces the Array and Color data type.


          		 Working with Bitmaps

          In this weeks tutorial we will look at using Bitmaps and SVG files in Processing.  We will look at how we can access the colour of each pixel, and how we can slice up a bitmap to create new images, and how we can create animated "sprites" with image sequences.


          		    
          W5 Sound processing and Visualisation

          This week's lecture will focuses on working with digital audio using the Minim library.


          		 Sound and Image

          W6 Working with text

          This week covers working with text as char and String data types, and using PFont to display text on the canvas.

          In this weeks tutorial we will look at working with Fonts and Text in Processing. This is the last  in class activity that will form part of the assignment 1 submission.

        • Resource Materials

          There are tons of great Processing resources online, more than can possibly be summarised in this single page. The best starting place is the Processing web site itself: http://processing.org

          The following book is recommended to support DXB303:

          Reas, C. and B. Fry (2007) Processing : a programming handbook for visual designers and artists. Cambridge, MA: MIT Press.

          However, there is no need to purchase this text as the library provides access an electronic version. AND more importantly there is a wealth of support materials available online.

          The processing web site has a number of excellent tutorials which we will use to support tutorials. And there the examples pages is a good start, and we use several of these in class.  The forum is also good places to find additional support.

           

                 

          Other references include:

          • Bohnacker, H., B. Gross and J. Laub, (2012) Generative Design: Visualize, Program, and Create with Processing. Princeton Architectural Press, New York
          • Fry, B. (2008) Visualizing data. Sebastopol, CA: O’Reilly Media, Inc
          • Maeda, J. (1999) Design by numbers. Cambridge, MA: MIT Press
          • Maeda, J. and Massachusetts Institute of Technology Media Laboratory (2004) Creative code. London: Thames & Hudson
          • Maeda, J. (2006) The laws of simplicity. Cambridge, MA: MIT Press
          • For other text books see :  https://www.processing.org/books
